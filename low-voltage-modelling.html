Low-Voltage Level Power System Modelling

Forecasting energy consumption and maximum demand for residential and small commercial systems using time series data from Southern Germany.

View on GitHub Live Demo (Coming Soon!)

Project Overview

This project forecasts energy consumption and maximum demand at the low-voltage level based on detailed time series data from 11 households in Southern Germany between 2014 and 2020. The data includes 15- and 60-minute interval measurements covering overall consumption, photovoltaic generation, electric vehicle charging, and appliance-level usage. The goal is to understand load patterns and build regression models to predict peak demand, aiding energy efficiency and grid stability efforts.

Data Sources

Primary Dataset

  • Open Power System Data: Comprehensive time-series data from 11 households including consumption, PV generation, EV charging, and appliance loads with 15- and 60-minute resolution.
    https://open-power-system-data.org/

Notes

The data covers several years and multiple load types but is geographically limited to Southern Germany.

Methodology

  1. Data Preparation and Cleaning
    • Standardized columns and filtered to Dec 2015–Jun 2017 period.
    • Handled missing values and aggregated PV generation data.
  2. Feature Engineering and Dimensionality Reduction
    • Performed correlation analysis to select relevant features.
    • Applied Principal Component Analysis (PCA) retaining 90% variance.
    • Split data into training and test sets with an 80/20 ratio.
  3. Model Training and Evaluation
    • Trained multiple regression models: Linear, Ridge, Lasso, Elastic Net, KNN, Decision Tree, Random Forest, Gradient Boosting.
    • Evaluated models using R², MAE, and RMSE metrics.

Results Summary

  • Gradient Boosting, KNN, and Random Forest models achieved R² ≈ 0.99, showing excellent predictive accuracy.
  • PCA improved model robustness by reducing dimensionality and overfitting risk.
  • Appliance-level consumption strongly correlated with maximum demand.

Future Work

  • Develop real-time monitoring dashboards for stakeholders.
