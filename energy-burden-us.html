Quantifying Energy Burden in the United States

A State-Level Assessment of Electricity Affordability and Climate Impact (2021–2023)

View on GitHub Read Methods

Project Overview

This project investigates the energy burden — the percentage of household income spent on electricity — across U.S. states from 2021 to 2023. It merges monthly electricity price and consumption data from the U.S. Energy Information Administration (EIA), annual median household income from the American Community Survey (ACS), and monthly climate metrics (temperature, Cooling Degree Days) from NOAA.

Data Sources

Primary datasets

  • EIA: Monthly residential electricity prices, sales (MWh), and customer counts. (EIA — Electric Sales, Revenue, and Price).
  • ACS: Annual median household income by state (ACS 1-Year Estimates). (data.census.gov).
  • NOAA (NCEI): Monthly average temperature and Cooling Degree Days (CDD) via Climate at a Glance API.

Coverage & notes

The combined panel covers all 50 states + DC for 2021–2023. EIA and NOAA are monthly; ACS income is annual and is aligned to monthly records during preprocessing (income broadcast to each month for the same year).

Methodology

  1. Data Collection & Preprocessing
    • EIA & ACS: direct download and CSV ingestion.
    • NOAA: programmatic download via NCEI Climate at a Glance API (requires API key).
  2. Data Cleaning & Preparation
    • Standardize column names, normalize state naming, create state_name and state_code fields, coerce numeric types.
    • Filter to target years (2021–2023) and handle missing climate values (notably Alaska/Hawaii CDD/HDD gaps) with transparent rules.
    • Convert EIA MWh & cents units to kWh / dollars as needed for calculations.
  3. Feature Engineering & Derived Metrics
    • Compute avg_monthly_consumption_kwh_per_customer = (sales_mwh * 1000) / customers_count.
    • Compute avg_monthly_cost_usd = avg_monthly_consumption_kwh_per_customer * (price_cents_per_kwh / 100).
    • Aggregate monthly costs to annual costs for each state and compute Energy Burden (%) = (annual_electricity_cost / median_household_income) * 100.
  4. Exploratory Data Analysis (EDA)
    • Visualize distributions (price, CDD, consumption), seasonal trends (monthly averages), and geographic patterns (choropleth/heatmaps).
    • Compute correlation matrices and inspect bivariate relationships to inform regression variables.
  5. Statistical Modeling & Interpretability
    • Run OLS regression on annual metrics (energy burden vs. income, price, temp, CDD).
    • Use SHAP to interpret feature contributions for the fitted model.

How to Run the Project

Clone the repository and change directory:

git clone https://github.com/sayalisnage/energy-burden-us.git
cd energy-burden-us

Create a virtual environment (recommended) and activate it:

python -m venv venv
# macOS / Linux
source venv/bin/activate
# Windows (PowerShell)
venv\Scripts\Activate.ps1

Install dependencies:

pip install -r requirements.txt

Acquire Raw Data

  • EIA (CSV) — download from the EIA site and place files in data/raw/eia/.
  • ACS (CSV) — download the ACS tables used and place files in data/raw/acs/.
  • NOAA (API) — obtain an NCEI API key and place it in a .env file: 
    NOAA_API_KEY=your_api_key_here
    Then run the climate acquisition script: 
    python scripts/data_acquisition/noaa_data_acquisition.py

Preprocess & build final datasets (example):

python scripts/noaa_data_preprocessing.py
python scripts/data_preprocessing.py

After preprocessing, processed files will be saved to data/processed/. Open the analysis notebook to reproduce EDA, calculations, and modeling:

jupyter notebook notebooks/energy_burden_analysis.ipynb

Results Summary

  • Southern states experience higher energy burdens driven by a combination of lower median incomes and high cooling demands.
  • Median household income is the strongest negative driver of energy burden (higher income → lower burden).
  • Electricity price and climate metrics (average temperature, CDD) are significant positive drivers of burden.

Future Work

  • Expand the time window (5–10 years) for trend stability and policy analysis.
  • Integrate housing characteristics (age, insulation, AC penetration) for deeper equity analysis.
  • Build predictive models to forecast energy burden under alternate climate and economic scenarios.
