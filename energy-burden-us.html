Quantifying Energy Burden in the United States

A State-Level Assessment of Electricity Affordability and Climate Impact (2021–2023)

View on GitHub

Project Overview

This project investigates the percentage of household income spent on electricity (energy burden) across the United States from 2021 to 2023, combining data from EIA, ACS, and NOAA to analyze regional disparities, drivers, and policy implications.

Data Sources

  • EIA: Monthly residential electricity prices, sales, and customer counts (link).
  • ACS: Annual median household income per state (link).
  • NOAA: Monthly climate data (temperature & Cooling Degree Days) (link).

Methodology

  • Data collection and preprocessing
  • Cleaning and alignment of datasets
  • Calculation of energy burden %
  • EDA to identify regional patterns
  • Statistical modeling and SHAP interpretability

    • Technology Stack

    • Python, Pandas, NumPy
    • Matplotlib, Seaborn, Plotly
    • Scikit-learn, Statsmodels, SHAP
    • Jupyter Notebooks, Git

    How to Run the Project

    1. Clone the repository: 
      git clone https://github.com/sayalisnage/pm25-air-quality-forecasting.git
cd energy-burden-us
    2. Create a virtual environment (recommended) 
      python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
    3. pip install -r requirements.txt
    4. Acquire EIA, ACS, and NOAA data (see README for details).
    5. Run preprocessing scripts and Jupyter notebook analysis.

    Results Summary

    • Southern states face highest burdens due to low incomes and high cooling needs.
    • Income is the strongest negative driver of burden.
    • Price and climate metrics are significant positive drivers.
