Weather Forecast Application

A command-line application to fetch and display local weather forecasts in a user-friendly format.

View on GitHub

Project Overview

This project is a command-line weather forecasting application written in Python. It's a practical demonstration of using external APIs to retrieve and process data, implementing robust error handling, and designing a user-friendly console interface. The application allows users to get up-to-date weather information for any U.S. location by entering a ZIP code or a city and state, and customize the temperature display to their preference.

Features

  • **Command-Line Interface:** Provides a text-based interface for easy interaction in a terminal.
  • **Location Flexibility:** Supports lookups by both ZIP code and city/state.
  • **Multi-Unit Temperature:** Users can choose to view temperatures in Fahrenheit, Celsius, or Kelvin.
  • **API Integration:** Uses the OpenWeatherMap API for accurate, real-time weather data.
  • **Robust Error Handling:** Designed with `try-except` blocks to handle connection issues, invalid API responses, and bad user input.
  • **Readable Output:** Presents the weather data in a well-formatted and easy-to-read manner.

Technology Stack

  • Programming Language: Python
  • Libraries: `requests`
  • API: OpenWeatherMap
